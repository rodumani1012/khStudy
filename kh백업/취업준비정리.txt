
취업 전
- 하루 1시간 이상은 자소서 쓰기 : 최대한 많이 쓰고, 계속 고치고, 
  입사하지 않을 회사에도 면접은 가기
- 취업사이트(사람인 등) 하루 한번 이상 보기
- 집에만 있지 말기 : 카페에서 공부하는 것도 좋고, 산책도 좋고, 친구 만나는 것도 좋음. 
  하루종일 집에만 있으면 정신적으로 버티기 힘드니 하루 한번정도는 바깥바람 쐬기
- 뉴스 보기. 요즘 이슈 등을 면접에서 물어보는 경우도 있음(면접관이 분위기 전환하려고)

자소서 작성법
1) 최대한 많은 문항을 모아와서 '통합자소서'를 만든다.
2) 최대한 길게 쓴다.
3) 공통되는 문항들을 '공통자소서'로 작성한다.
4) 원하는 회사의 인재상에 맞춰서 내용을 수정한다. (창의, 열정)

신입 때
- 절대 지각하지 않기. 출근 시간 10분 정도 전에 도착해서 미리 준비하는게 
  좋은 인상 줄 수 있음.
- 업무가 주어졌을 때 막히는 부분이 있다면 질문하는 방법(2~3시간 고민 후 가기)
  1) 문제에 대해 정리한다. (어느어느 부분에서 데이터가 넘어가지 않습니다.)
  2) 해결하려고 했던 방안을 정리한다. (해결하기 위해 A,B,C를 시도해 봤습니다.)
  3) "해결해주세요."가 아닌 "제가 어떤 방법을 더 시도해 봐야 할까요?" 라고 말하기.
- 회사가 마음에 안들 때 : 자소서 많이 쓰기. 1~2년 미만인 경우 중고 신입으로 
  들어가는 방법도 있고, 경력이직도 아예 없는 것은 아님.
- 퇴근 후, 주말 등 자투리 시간에 자기계발 하기
  최근 IT 쪽 트렌드 빅데이터, IoT, 머신러닝, 클라우드, 블록체인
  1) 빅데이터 개발 : (linux) hadoop, spark 등
  2) 빅데이터 분석 : r, python
  3) IoT : 모바일 어플리케이션(java, kotlin, swift 등), 아두이노
  4) 머신러닝 : c, python
  5) 클라우드 : aws, azure
  6) 블록체인 : 이더리움(솔리디티) 등
- 고졸, 초대졸인 경우 방통대 등으로 4년제 학위 따기(연봉이 다른 회사들이 있음)
  방통대 같은 경우, 재학 중 정보처리기사 따면 졸업논문 대체 되는것으로 알고있음
