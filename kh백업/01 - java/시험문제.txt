패키지
-서로 관련있는 클래스들을 묶음
-클래스명이 같아도 패키지가 다르면 생성 가능
- 클래스의 실제 이름은 패키지명 포함

java의 특징
- 멀티쓰레딩, 동적 로딩, 메모리 자동 관리, 플랫폼에 독립적

ex) String s = "a";
{ s = "ab" } public String ex(String s) { this.s = s;}
instance 변수의 메모리 할당 순서 : jvm 기본값(null) -> 명시적 초기값(a) -> 초기화 블록(ab) -> 매개변수 있는 생성자(this.s = s;)

type(크기)

연산자(우선순위, 증감연산자, 삼항연산자 중첩)

제어문

접근제한자 public(+,동그라미) protected(#,다이아) default(~, 세모) private(-,네모)

String 의 특징

overloading(파라미터의 갯수나 파라미터의 타입이 달라야하고 리턴타입은 상관없다)

배열(선언, 값 넣기, 값들의 합/평균, 랜덤 값들의 최대값/최소값, 객체 배열ex)Score[], People[] 등)

구구단

class/instance만들기(getter/setter, 객체 생성, 객체 배열,...) ex) +getSum(int); 퍼블릭에 파라미터 하나 받는 getSum메소드

Scanner로 입력받은 값과, 클래스 내부의 값 비교

소수(1과 자기 자신으로 밖에 나눌 수 없는 수) 구하기(1 2 3 5 7 11...)
